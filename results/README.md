## Reproduction

```bash
# From repo root — regenerate all outputs:
python examples/run_full_scenario.py        # Simulation figures + JSON
python run_simulation.py                     # Reference trajectory figures
python run_phase2_extractions.py             # Phase-2 sweep figures + JSON

# From experiments/tcga/ — regenerate TCGA figures:
python tcga_multicancer.py
python tcga_tumor_normal.py
python tcga_immune_subtype.py
python tcga_survival.py
python tcga_stage_analysis.py
python generate_manuscript_figures.py
python sensitivity_threshold_sweep.py
python sensitivity_rmst.py
python sensitivity_immune_residualization.py
python composite_observability_index.py
python run_external_validation.py

# Sync and provenance:
python sync_manuscript_artifacts.py --rewrite-tex
```

All outputs are model-derived and illustrative; they are not clinical predictions.
